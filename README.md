# AddApp — ADHD Super App

Your personal ADHD management super app. Built with React + Vite.

## Features (v1)
- ⚡ Routines — step-by-step guided routines with timers, skip, and defer

## Coming soon
- ✅ Gamified task board
- 📅 Calendar
- 💚 Health & mood log
- 📓 Journal

## Deploy

This app is deployed automatically via Vercel when you push to GitHub.

## Local development

```bash
npm install
npm run dev
```
